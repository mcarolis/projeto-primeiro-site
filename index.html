O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas da web. Ela é a linguagem que os navegadores da web usam para interpretar e renderizar o conteúdo das páginas da web. HTML permite que você estruture o conteúdo de uma página da web, incluindo texto, imagens, links, formulários e outros elementos, para que os usuários possam acessar e interagir com informações online.

A principal característica do HTML é sua capacidade de usar "tags" para marcar elementos em uma página e definir sua estrutura e significado

O que são as Tags no HTML?

As tags no HTML são elementos usados para marcar e estruturar o conteúdo de uma página da web. Elas são essenciais para definir como o navegador da web deve exibir e interpretar o conteúdo. As tags HTML geralmente são compostas por um nome de elemento colocado entre colchetes angulares ("<" e ">"). As tags HTML são usadas em pares, com uma tag de abertura e uma tag de fechamento, e o conteúdo a ser afetado é colocado entre essas tags. A tag de fechamento é semelhante à tag de abertura, mas inclui uma barra ("/") antes do nome do elemento.

Exemplos de tags no HTML e suas funções:

  • <html>: A tag que envolve todo o conteúdo da página

  • <head>: Contém informações sobre a página, como título, metadados e links

  • <title>: Define o título da página, que é exibido na barra de título do navegador

  • <body>: Contém o conteúdo visível da página, como texto, imagens e links

  • <h1 ao h6>: São usados para criar cabeçalhos com diferentes tamanhos

  • <p> : Cria parágrafos de texto

  • <ul> e <ol> :Usadas para criar listas não ordenadas (com marcadores) e listas ordenadas (com números ou letras)
